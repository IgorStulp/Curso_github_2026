cd .. -> pasta acima/volta para a pasta anterior
ls -> lista tudo que tem na pasta
pwd -> vejo onde estou
mkdir -> cria pastas
ls -a -> mostra os arquivos ocultos
cd (pasta + /) -> move entre as pastas
cat -> mostra o que tem dentro dos arquivos
rm - rf -> exclui diretório
rm -> remove pastas do diretório
touch -> cria arquivos sem nada dentro

###########################################################

git init . -> cria um git na pasta atual
git add -> adiciona uma pasta no staged
git commit -m "msg" -> cria um commit com as pastas do staged
git status -> vê o status do git
git reset (pasta) -> remove uma pasta do staged
git diff -> verifica o que foi mudado no arquivo
git log -> vê o log dos commits
git checkout -b (nome) -> cria uma nova branch
git checkout (nome) -> vai para outra branch
git branch -> mostra as branchs do meu projeto
git branch -d (nome) -> exclui uma branch
git merge (nome da branch) -> mescla as informações das branchs
git pull -> recebe coisas do origin
git push -> envia coisas para o origin
git clone -> clona
git fetch -> traz todas branchs do (remoto)

###########################################################

docs -> apenas mudanças de documentação
feat -> uma nova funcionalidade
fix -> correção de um bug
perf -> mudança de código focada em melhorar performance
refactor -> mudança de código que não adiciona uma funcionalidade e também não corrigi um bug
style -> mudanças no código que não afetam seu significado 
(espaço em branco, formatação, ponto e vírgula, etc)
tes -> adicionar ou corrigir testes
